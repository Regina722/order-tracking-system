# Order Tracking System

Backend system for tracking orders from creation to warehouse arrival,
boxing, delivery and archiving.

## Project goal
This is a learning project focused on backend logic, state transitions
and clean domain modeling.

## MVP scope
- Create order cards with tracking information
- Mark orders as arrived at warehouse
- Group orders into boxes
- Track box delivery
- Automatically archive orders after box delivery

## Tech stack
- Java 21
- Maven
- (Spring Boot – later)
- (Database – later)

## Project status
MVP core in progress.
